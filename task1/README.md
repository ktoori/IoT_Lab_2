
### Описание алгоритма
Множество Мандельброта – это множество комплексных чисел c, для который последовательность z(0) = 0, z(n+1) = z(n)^2 + c остаётся ограниченной при n, стремящемуся к бесконечности.

### Принцип работы алгоритма
1.	Берётся точка c в комплексной плоскости.
2.	Для неё инициализируется комплексное число z = 0;
3.	До заданного максимального числа итераций выполняется:
a.	Вычисление z = z2 + c.
b.	Если |z| > 2, точка не принадлежит множеству.
4.	Если после максимального числа итераций |z| <= 2, точка принадлежит множеству мандельброта.
5.	Алгоритм повторяется аналогично для остальных точек в комплексной плоскости.

### Описание реализации параллельной программы
- Для работы с комплексными числами были созданы две структуры данных: Complex, отвечающая за представление комплексных чисел, и Point, предназначенная для хранения информации о точке (её координаты и принадлежность к множеству Мандельброта).
- Функция mandelbrot вычисляет количество итераций для определения принадлежности точки c к множеству Мандельброта:
	1. Инициализируется z = 0 + 0i.
	2. Итеративно вычисляется z = z2 + c.
	3. Проверяется условие |z| > 2.
	4. Возвращается количество выполненных итераций.
- Функция main – основная функция, в которой задана область вычислений для координат комплексных чисел и в которой осуществляется вызов функции mandelbrot для каждой точки. Также для измерения времени работы алгоритма берётся усредненное значение на 20 итерациях.
- Результаты с точками, входящими во множество Мандельброта записываются в файл mandelbrot.csv.
